 * * * * * 🩸 BloodConnect - Blood Donation Platform Server


Backend API for the BloodConnect  Blood Donation Platform built with Node.js, Express.js, and MongoDB.


* * * Features

* JWT Protected APIs
* Better Auth Token Verification
* Role-Based Authorization (Admin, Volunteer, Donor)
* User Management APIs
* Blood Donation Request CRUD
* Funding Management APIs
* Pagination Support
* Secure MongoDB Connection
* Environment Variable Configuration



* * * Tech Stack

* Node.js
* Express.js
* MongoDB
* JOSE (JWT Verification)
* CORS
* Dotenv



* * *  Packages Used

* express
* mongodb
* jose-cjs
* cors
* dotenv
